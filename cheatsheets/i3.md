## focus
```j``` left

```k``` down

```l``` up

```;``` right

```spacebar``` toggle

## change container layout
```e``` tiling

```s``` stacking

```w``` tabbed

## misc
```shift spacebar``` toggle floating

```shift c``` reload config file

```shift r``` restart i3 inplace
